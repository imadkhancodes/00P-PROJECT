Blood Bank Management System 
A simple console-based Blood Bank Management System developed in C++ using Object-Oriented Programming (OOP) concepts.
This project was created as a semester project for the Object-Oriented Programming (OOP) course.
Features
 •	Add Donor Information
 •	Display Donor Records
 •	Add Recipient Information
 •	Display Recipient Records
 •	Search Donor by Blood Group
 •	Process Blood Requests
 •	Manage Blood Units
 •	Menu-Driven System

Blood Bank Management System 
A simple console-based Blood Bank Management System developed in C++ using Object-Oriented Programming (OOP) concepts.
This project was created as a semester project for the Object-Oriented Programming (OOP) course.
Features
•	Add Donor Information
•	Display Donor Records
•	Add Recipient Information
•	Display Recipient Records
•	Search Donor by Blood Group
•	Process Blood Requests
•	Manage Blood Units
•	Menu-Driven System

Project Description
The Blood Bank Management System helps manage donor and recipient records efficiently. The system allows users to store donor information, search blood groups, and process blood requests in a simple and organized way.
The project was developed to understand and implement Object-Oriented Programming concepts in a real-world application.
Objectives
•	Automate blood donor management
•	Reduce manual record handling
•	Improve blood searching process
•	Apply OOP concepts practically
•	Build programming and problem-solving skills
Limitations
•	Console-based application
•	No database connectivity
•	No GUI support
•	Limited storage capacity
